# Tools

Utilitários auxiliares que não são o indicador nem pertencem a uma pesquisa específica — tipicamente:

- **Exportadores MQL5**: scripts para extrair dados do MT5 (barras, valores de buffers do IFM) e alimentar as pesquisas em Python.
- **Scripts de apoio reutilizáveis**: verificação de paridade indicador ↔ Python, conversores de formato, etc.

Cada ferramenta deve ter um comentário de cabeçalho (ou seção aqui) dizendo o que faz, como usar e o que produz.
