# Variantes

Linhas de desenvolvimento do IFM que **não** substituem a versão principal (`src/IFM.mq5`) — experimentos direcionais, forks de comportamento, versões especializadas.

Convenção:

- Nome com sufixo identificador: `IFM-X.mq5`, `IFM-Y.mq5`, ...
- Cabeçalho do arquivo explica no que difere da principal e por quê.
- Se uma variante for promovida a versão principal, ela substitui `src/IFM.mq5`, ganha tag + entrada no CHANGELOG, e o arquivo da variante é removido daqui.
